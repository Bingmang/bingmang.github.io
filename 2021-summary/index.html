2021 年度总结

Posted on 2022-01-04

时间点

先来说几个今年的时间点：

  • 1月18日 寒假回家
  • 3月16日 被实验室派去福州出差
  • 5月25日 回京
  • 7月5日 面试 SmartX
  • 7月19日 入职 SmartX 实习生
  • 10月11日 SmartX Offer

今年看了哪些书

今年看的书不多，也几乎没有看村上春树的小说，只记得重启了好多次《1Q84》但还是没能坚持看下去，可能是因为自己看不懂那层隐喻，且剧情又不够科幻导致的。科幻小说尝试过《星之继承者》和《银河帝国》，也没有坚持看下去。觉得自己真是缺乏坚持的能力。以下是今年尝试看过的书（或网课）：

小说类：

  • 1Q84
  • 星之继承者
  • 银河帝国
  • 动物庄园
  • 直到世界尽头
  • 败者为王

技术类：

  • MIT 6.824
  • Redis 设计与实现
  • 高性能 MySQL
  • 正则表达式必知必会
  • SQL 必知必会
  • 深入浅出 Prometheus
  • Kubernetes in Action
  • 极客时间：Linux 性能优化实战
  • 深度实践 KVM
  • SRE：Google 运维解密
  • 深入剖析 Kubernetes
  • DDIA
  • 博客：网络基本功系列
  • 100 Go Mistakes

工具类：

  • 零挫败感人生
  • 富爸爸穷爸爸
  • 亲密关系
  • 现代西班牙语
  • 睡眠革命
  • 刻意练习

然而看完的只有《直到世界尽头》和《败者为王》，因为那个时候刚好很喜欢羽毛球，而且读这类自传书不需要做笔记之类的，没有思想负担。其实感觉看其他书也应该换种模式，如果思想负担太重，真的很难坚持看完一本书。

感觉看书真的是一口气的事情，“一鼓作气，再而衰，三而竭”的意思吧。

2021 我推荐的三本书：深度实践 KVM（虚拟化技术 overview，没有技术细节，读起来很通畅）、败者为王（李宗伟自传，我很喜欢李宗伟）、睡眠革命（睡满 8 小时只是谬论，醒了后睡懒觉没啥用）

今年的兴趣爱好

发现每年的12月是玩心最重的一个月，去年12月去了潜水和滑雪，今年12月一直在打羽毛球（平均两三天一场），也是打算好好培养一下自己体育方面的兴趣爱好。

在羽毛球上做了不小的投入，拍子、鞋、袜子、球、护膝、防滑粉、陪练，技术确实有些长进，但还是差太远了，现在真是极其业余，希望能趁寒假在家里有机会学一学。

今年去哪儿玩了

3月12日-3月15日和老妈一起去了杭州和上海玩，总共消费 3613 元；

今年值得开心的事情

没怎么投简历和面试，投了字节二面挂之后就投了 SmartX，孤注一掷最后转正，拿到了满意的薪资，做的事情也比较符合自己的兴趣。

学习羽毛球，保持锻炼身体。

Summary

总的来说，今年过的挺平淡的，除了找到一份满意的工作外，其他的事情都平平淡淡，感觉今年过的不是很有意义。赚不到什么钱，爱情也不见起色，身边也没有知心的朋友。

2022 年的愿望

把更多精力放到社交和羽毛球中，交到知心的好朋友，多出门，改改自己内向的性格。

# 年度总结
虚拟机 IO 慢速导致挂载超时进入 emergency mode 问题排查
